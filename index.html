pwaxMask

From -0 to Big.js: all your numeric input edge cases covered.

This is just a small showcase of what pwaxMask can do!

1) Basic number

no attributes
class: '.dec'
data-digits="4"
data-accept-both-decimal="true"

2) Currency

class: '.eur'
class: '.usd'
class: '.inr'
class: '.chf'
class: '.chf'
class: '.jpy'
class: '.gbp'

3) Percent / Units

class: '.per'
class: '.per', data-digits="2"
data-suffix=" ‰" data-digits="3" data-max="1000"
data-unit="bp"
data-prefix="A_LONG_PREFIX "
data-suffix=" is a Magic Number" data-prefix="Number "

4) Grouping styles

data-group-style="standard"
data-group-style="indian"
data-prefix="¤ " data-decimal="," data-group="’" data-digits="2" data-group-pattern="4,4"

5) Signs & styles

data-allow-negative="true" data-show-zero-sign="true"
data-allow-negative="true" data-minus-behavior="toggle"
data-allow-negative="true" data-negative-style="parens"
data-allow-negative="true" data-negative-style="parens" data-negative-parens="<,>"
data-allow-negative="true" data-negative-sign-symbol="⊖"
data-show-positive-sign="true" data-positive-sign-symbol="⊖"

6) Clamp, min/max & step

data-min="0" data-max="100" data-step="0.25" User arrow up, down
data-min="0" data-max="100" data-step="0.5" data-clamp-mode="blur" data-snap-to-step="true"

7) Raw on focus / Select

data-select-on-focus="true" data-select-decimals="true"
data-allow-wheel="true" data-hold-accel="true" data-step="0.25" data-min="0" data-max="9999"

8) Empty handling

data-keep-empty="true"
data-blank-zero="true"

9) Digit normalization

Arabic/Persian digits (۱۲۳۴۵۶۷۸۹۰) → ASCII

10) Big.js

data-decimal="." data-group="," data-digits="18" data-prefix="Ξ " data-allow-wheel="true" data-step="0.000000000000000001" data-math-engine="big" inputmode="decimal"
